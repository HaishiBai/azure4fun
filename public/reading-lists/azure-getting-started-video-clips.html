Azure Reading List - Getting Started (video clips)

This reading list contains a number of hand-picked online video clips for you to get started with Azure. Azure is a dynamic platform that keeps a rapid pace in innovation. Hundreds of new features and improvements are being released every year. There are abundant resources online to study Azure, however sometimes it could be overwhelming to pick a cohesive set of contents to read. The goal of this reading list is provide a concise, and up-to-date reading list to get you on your journey to become an Azure expert.

The goal of this reading list is to prepare you for real development work. After covering some basic concepts, this list focuses on hands-on contents that can guide you through the basic steps to kick off your own projects. It’s recommended that you get an Azure subscription and follow the steps as you watch the video clips to gain first-hand experiences with Azure.

Fundamentals

  • Microsoft Azure Overview   01:39:48

    This Microsoft Virtual Academy video features Scott Guthrie, who not only explains key value propositions and fundamental concepts of Azure, but also provides step-by-step guidance of getting started with Azure, including provisioning Virtual Machines, Virtual Networks, SQL Database, Web Sites, Storage as well as Azure Active Directory.
    Please note the following changes since the video was recorded:
    • Windows Azure has been rebranded to Microsoft Azure.
    • A new Azure Portal, http://portal.zure.com is available for preview now.
    • Web Sites has been renamed to Web Apps, which is part of a new integrated App Service.
  • Microsoft Azure IaaS Technical Fundamentals   00:48:00

    This Microsoft Virtual Academy video features Mark Russinovich, who gives an overview of recent Azure developments (up to December 2014) in areas of IaaS, identity, and application services. He thoroughly explains how Azure differentiate from its competitors by providing hyper-scale, enterprise grade and hybrid solutions.

Architecture

This section consists of a couple of video clips that cover a number of commonly seen scenarios on cloud, including batch processing, long-running services, IoT, and media streaming etc. It’s quite a jump from the previous sections and you’ll need some prior architecting experiences to fully understand the episodes. Regardless, it’s important to get you mind wrapped around cloud characteristics and make sure your applications are designed for cloud in order to take advantages of the Quality of Services (QoS) opportunities provided by cloud. These video clips touch upon a large number of Azure services such as Service Bus and Media Services, however there are no detailed introductions of these services. You can find more details on these services at azure.com.
