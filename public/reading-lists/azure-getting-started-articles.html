Azure Reading List - Baseline

This reading list contains a number of hand-picked online resources for you to get started with Azure. Azure is a dynamic platform that keeps a rapid pace in invotaion. Hundreds of new features and improvements are being released every year. There are abundant resources online to study Azure, however sometimes it could be overwhelming to pick a cohensive set of contents to read. The goal of this reading list is provide a concise, and up-to-date reading list to get you on your journey to become an Azure expert.

Many of the picked articles are from azure.microsoft.com, which is the official site of Microsoft Azure. There are hundreds of articles, which are contributed by the people who designed and implemeneted various Azure services.

Internet of Things

Internet of Things (IoT) calls for an end-to-end data pipeline from data generation and collection, to transformation and storage, and to analytics and presentation. Scalability and performance are two keys to successful IoT solutions. And Azure provides a comprehensive solution that enables various IoT scenarios at scale.

  1. Event Hubs Overview

    Event Hubs is a cloud-scale telemetry ingestion from websites, apps, and devices.

  2. Stream Analytics Overview

    Stream Analytics is a real-time stream processing service.

Networking

Azure Virtual Network is an important technology to enable many hybrid scenarios, where on-premises servers and virtual machines on cloud can connect and communicate with each other as if they were in a same local network. Virtual Network also allows you to create virtualized networks on cloud to simulate a local network that provides connectivity among your Azure virtual machines.

  1. Virtual Network Overview

    Provides a private network on cloud.

  2. VPN Gateway

    Bridges cloud virtual networks and your on-premises infrastructure
